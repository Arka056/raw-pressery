# 🥭 Raw Pressery – Mango Juice Web Application

A modern and refreshing **Mango Juice Web App** that showcases fresh juice products with a clean and attractive interface.  
This project focuses on **frontend development and UI design**, inspired by premium juice brands.

---

## 🌟 About The Project

**Raw Pressery** is a simple web application designed to promote fresh mango juice products through an engaging landing page.

The goal of this project is to demonstrate:

- Clean UI design
- Basic frontend development
- Product presentation layout
- Responsive web structure

---

## 🚀 Features

- 🥭 Beautiful mango juice themed landing page
- 🎨 Clean and minimal UI design
- ⚡ Fast and lightweight website
- 📱 Responsive layout
- 🍹 Product showcase section

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
